# 函数属性和方法

ECMAScript 中的函数是对象，所以有属性和方法。每个函数都有两个属性，length 属性保存函数定义的命名参数的个数，prototype 指向原型对象（在第八章详细介绍过了）。

函数还有两个方法：apply() 和 call()，都会以指定的 this 值来调用函数

* apply() 接收两个参数：函数内 this 的值和参数数组。
* call() 的第一个参数也是函数内 this 的值，但是接下来的参数是以逗号分隔的参数列表（参数逐个传递）。

```javascript
function myCall(thisArg, ...args) {
    Function.prototype.apply(thisArg, args)
}
```

使用 apply() 和 call() 的好处是可以将任意对象设置为任意函数的作用域，这样对象可以不用关心方法。

ES5出于同样的目的定义了一个新方法 bind()，它会创建一个新函数实例，其 this 值会被绑定到传给 bind() 的对象。
