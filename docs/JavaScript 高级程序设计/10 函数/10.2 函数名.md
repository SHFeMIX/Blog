# 函数名

**函数名就是指向函数的指针**

**ECMAScript 的所有函数对象都会暴露一个只读的 name 属性，字符串。没有名称就是空字符串，Function 构造函数创建的是‘anonymous’**

```javascript
function foo() {}
let bar = function() {};
let baz = () => {};

console.log(foo.name);   // foo
console.log(bar.name);   // bar
console.log(baz.name);   // baz
console.log((() => {}).name);   // ''
console.log((new Function()).name);  // anonymous
```

**get，set 函数或者使用 bind() 实例化，标识符会带前缀**

```javascript
function foo() {}

console.log(foo.bind(null).name);   // bound foo

let dog = {
    years: 1,
    get age() {
        return this.age;
    },
    set age(value) {
        this.age = value;
    }
};

function foo() {}
let propertyDescriptor = Object.getOwnPropertyDescriptor(dog, 'age'); console.log(propertyDescriptor.get.name); // get age 
console.log(propertyDescriptor.set.name); // set age
```
